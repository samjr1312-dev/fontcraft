## 🛠️ How to Use

1. **Paste or type** your text in the input box.
2. **Browse the font styles** – scroll through 100+ options and tap any style.
3. **Customize** – change text color, background color, font size, or toggle bold/italic.
4. **Copy** – click the “Copy to Clipboard” button.
5. **Paste** anywhere – Instagram bio, WhatsApp, Twitter, TikTok, gaming profiles, etc.
